## Kenney RPG Audio (CC0 / Public Domain)

Background music tracks and sound effects are sourced from the
**Kenney RPG Audio** pack by [Kenney](https://kenney.nl),

### BGM mapping

BGM tracks are placed in `assets/audio/bgm/`. Each track is loaded as OGG with an
MP3 fallback for full browser coverage (Phaser picks the first supported format).

| Track ID | Scene / Region | Source file |
|---|---|---|
| `title` | Title screen | Kenney RPG Audio BGM |
| `localhost_town` | Localhost Town (starting area) | Kenney RPG Audio BGM |
| `pipeline_pass` | Pipeline Pass | Kenney RPG Audio BGM |
| `container_port` | Container Port | Kenney RPG Audio BGM |
| `cloud_citadel` | Cloud Citadel | Kenney RPG Audio BGM |
| `kernel_caves` | Kernel Caves | Kenney RPG Audio BGM |
| `serverless_steppes` | Serverless Steppes | Kenney RPG Audio BGM |
| `three_am_tavern` | 3am Tavern | Kenney RPG Audio BGM |
| `legacy_codebase` | Legacy Codebase area | Kenney RPG Audio BGM |
| `outcast_network` | Outcast Network (hidden) | Kenney RPG Audio BGM |
| `shadow_registry` | Shadow Registry (hidden) | Kenney RPG Audio BGM |
| `battle_incident` | Battle — incident encounter | Kenney RPG Audio BGM |
| `battle_engineer` | Battle — engineer encounter | Kenney RPG Audio BGM |
| `battle_cursed` | Battle — cursed encounter | Kenney RPG Audio BGM |
| `battle_throttlemaster` | Battle — ThrottleMaster boss | Kenney RPG Audio BGM |

### SFX mapping

SFX files are placed in `assets/audio/sfx/` organised by category:

| Category | Directory | Game events | Source |
|---|---|---|---|
| Battle | `sfx/battle/` | hit, critical, heal, status effects, SLA timers, shame, budget drain, reputation, encounter start | Kenney RPG Audio SFX |
| UI | `sfx/ui/` | cursor move, confirm, cancel, text blip, save | Kenney RPG Audio SFX |
| Overworld | `sfx/overworld/` | emblem earn/sparkle, door locked, level up | Kenney RPG Audio SFX |
