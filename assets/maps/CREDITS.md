# Map Tile Asset Credits

> **Canonical attribution:** `assets/tiles/CREDITS.md` is the authoritative credits file for
> all tileset sources, licences, and tile indices. This file summarises which tilesets each
> map type uses; refer to the canonical file for full licence text and per-tile tables.

## Placeholder tiles

All maps use `stub_tiles.png` as their base tileset — a 5-tile placeholder strip generated for
development. This remains the first tileset for all maps; biome-specific tilesets are appended
as a second entry.

---

## `/dev/null Void` (`dev_null_void`) — `void_tiles.png`

14-tile custom pixel art strip (672×48 px) generated by `scripts/generate-tilesets.js`.
All tiles are original works — **no external asset packs used**.

Tile descriptors: `assets/tiles/void_tiles.tsj`.

---

## `Deprecated Azure Region` (`deprecated_azure_region`) — `wasteland_tiles.png`

14-tile custom pixel art strip (672×48 px) generated by `scripts/generate-tilesets.js`.
All tiles are original works — **no external asset packs used**.

Tile descriptors: `assets/tiles/wasteland_tiles.tsj`.

---

## Village / Nature / Interior biome maps

Main and gym maps in the `village`, `nature`, and `interior` biomes use the
**Kenney RPG Urban Pack** (`tilemap_packed.png` upscaled 3× to 48 px) as their sole
tileset (no stub fallback). Dungeon-type children use the custom biome tilesets below.

| Source | Author | License | URL |
|--------|--------|---------|-----|
| **Kenney RPG Urban Pack** | Kenney | CC0 1.0 | https://kenney.nl/assets/rpg-urban-pack |

---

## Biome tilesets (village / dungeon / nature / interior)

Custom 32-tile 8×4 sprite sheets (384×192 px) generated by `scripts/generate-tilesets.js`.
Used by dungeon-type maps with a matching `biome` field. All tiles are original works.

---

## All other regions

Tech regions use `kenney_tech_office.png` (50-tile custom pixel art, 240×480 px) generated by
`scripts/generate-tech-sprites.js`.  See `assets/tiles/CREDITS.md` for the per-tile breakdown.
