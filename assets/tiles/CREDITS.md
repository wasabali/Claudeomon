**Note:** The Kenney-sourced tiles above are currently rendered as pixel art in the
style of the Kenney RPG Urban Pack palette by `scripts/generate-tech-sprites.js`.
They should be replaced with the actual Kenney tiles when the pack is available by:
1. Downloading `tilemap_packed.png` from https://kenney.nl/assets/rpg-urban-pack
2. Upscaling 3× nearest-neighbour: `convert tilemap_packed.png -filter point -resize 300% upscaled.png`
3. Slicing the 14 tiles listed above at 48×48 px each
4. Dropping them into `assets/tiles/sprites/` with the matching `tile_NNN_name.png` filenames
5. Running `node scripts/compose-tech-tileset.js` (requires `npm install --save-dev canvas`)

### Upscale process

```bash
# Requires ImageMagick
convert kenney_tilemap_packed.png \
  -filter point -resize 300% \
  kenney_tech_office_raw.png
# Then manually slice the upscaled sheet into 48×48 tiles and place the
# Kenney-derived tiles listed above into the matching Cloud Quest grid slots.
# Once individual tile PNGs are ready, run scripts/compose-tech-tileset.js
# to stitch them into kenney_tech_office.png.
```
